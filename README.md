# Pardeep Hospital Management System

Pardeep Hospital is a modern, full-stack hospital management web application built with Next.js, React, and Tailwind CSS. It provides a comprehensive platform for managing hospital operations, including patient registration, doctor management, appointments, pharmacy, queue management, and more.

## Features

- **Landing Page**: Clean, modern landing page with navigation and hospital branding.
- **Admin Dashboard**: Secure admin portal for managing departments, doctors, patients, appointments, pharmacy, reports, notifications, and system settings.
- **Patient Dashboard**: Personalized dashboard for patients to manage appointments, view queue status, and access medical records.
- **Department Management**: Add, edit, and monitor hospital departments, including bed and occupancy stats.
- **Doctor Management**: Manage doctor profiles, statuses, and schedules.
- **Patient Management**: Register, search, and manage patient records and statuses.
- **Appointment Scheduling**: Book, view, and manage appointments with doctors.
- **Queue Management**: Real-time queue tracking for departments and doctors.
- **Pharmacy**: Manage pharmacy inventory and transactions.
- **Reports**: Generate and download hospital statistics and reports.
- **Settings**: Configure system preferences, notifications, and security settings.

## Tech Stack

- **Framework**: Next.js (React, TypeScript)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animation**: Framer Motion

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. **Open** [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` — Main application routes (landing, admin, dashboard, etc.)
- `components/` — Reusable UI and layout components
- `data/` — Static data for departments, doctors, events, etc.
- `context/` — React context providers (e.g., admin roles)
- `hooks/` — Custom React hooks
